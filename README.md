Sistema de Gestión de Países
link video: https://youtu.be/IV8xE-tkEow?si=2VyN3wgqEAnDJ4QY

Descripción

Este proyecto consiste en un sistema desarrollado en Python para gestionar información de países utilizando un archivo CSV como almacenamiento de datos.

El programa permite agregar, actualizar, buscar, filtrar y ordenar países, además de generar estadísticas sobre la información almacenada.




Tecnologías Utilizadas

- Python 3
- Módulo csv
- GitHub

Estructura del Proyecto

- "tp1_programacion.py": código principal del sistema.
- "paises.csv": dataset con la información de los países.
- "README.md": documentación del proyecto.

Ejecución

1. Descargar o clonar el repositorio.
2. Abrir una terminal en la carpeta del proyecto.
3. Ejecutar:

python tp1_programacion.py
